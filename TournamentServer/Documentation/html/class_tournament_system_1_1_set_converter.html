Detailed Description

JSONConverter for writing Sets to JSON.

There is no read method, as you should never be deserializing into a Set directly - use SetLinksReport for that, as my format uses Id references instead of nesting.

JsonConverter method used when serializing sets into JSON.

Uses Ids to replace nested references for entrants and other sets.

Currently, uses serialization of IWinnerDeciders by casting them into IWinnerDecider, and calling SetWinnerDeciderConverter methods. In the future, can either be replaced by a reflection search for an appropriate method to use instead given a custom class that implements IWinnerDecider, or expanding SetWinnerDeciderConverter code in this assembly.
