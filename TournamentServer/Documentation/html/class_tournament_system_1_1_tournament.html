◆ GetAllDataAsync()

async Task< Dictionary< string, string > > TournamentSystem.Tournament.GetAllDataAsync ( )

Method for getting the entire Data dict for further processing.

The actual returned value is a copy of the Data dict, not a reference to the actual Dict. This is for thread-safety, as we cannot guarantee that the Dict won't change after returning and we don't changes to the Dict to be reflected back into the tournament.

Returns
Dictionary{string, string} - the Data dictionary
